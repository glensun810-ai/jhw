# 待实现功能清单 - 基础设施与商业化

## 1. 后端基础设施
- **真实AI Judge模块**: 优化`AIJudgeClient`，使其能够更稳定、更准确地调用裁判LLM进行评估。
- **数据库结构优化**: `detailed_results` 和 `competitiveAnalysis` 等复杂数据目前作为JSON blob存储，不利于复杂查询和历史趋势分析。需要设计更规范的数据库表结构。
- **实时进度更新优化**: 考虑引入WebSocket或SSE，替代轮询机制。
- **AI API调用管理**: 针对AI平台的调用进行速率限制、成本管理和配额监控。
- **异步任务队列**: 引入Celery等任务队列，将耗时的AI调用从Flask主线程中解耦。

## 2. 商业化深化
- **订阅/支付模块**: 实现与微信支付等支付网关的集成。
- **用户角色与权限管理**: 针对企业版，实现多用户、多角色权限管理。
- **历史趋势分析**: 提供图表展示品牌GEO分数随时间变化的趋势。
- **竞品基准线**: 提供行业平均水平或自定义竞品组的基准线对比。
