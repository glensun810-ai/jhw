## 🚨 注意事项

### 1. 函数名称映射

在 `diagnosis_views.py` 中使用时，注意函数名称映射：

```python
# 导入时的别名
from wechat_backend.websocket_route import (
    send_diagnosis_progress as send_intelligence_update  # ← 使用别名保持兼容
)
from wechat_backend.services.realtime_push_service import (
    send_complete_sync as send_task_complete,  # ← 使用别名保持兼容
    send_error_sync as send_error  # ← 使用别名保持兼容
)
```

### 2. 函数调用保持兼容

由于使用了别名，现有代码无需修改：

```python
# 原有代码保持不变
send_intelligence_update(execution_id, data)  # 实际调用 send_diagnosis_progress
send_task_complete(execution_id, result)  # 实际调用 send_complete_sync
send_error(execution_id, error)  # 实际调用 send_error_sync
```
