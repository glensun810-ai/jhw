### 根本原因

**错误信息:**
```
RuntimeError: Working outside of application context.
This typically means that you attempted to use functionality that needed
the current application.
```

**错误位置:**
```
File "wechat_backend/views/diagnosis_views.py", line 751
    app_instance = current_app._get_current_object()
```

**原因分析:**

1. **`current_app` 是 Flask 的 context-local proxy**
   - 只能在请求线程中访问
   - 背景线程中访问会抛出 `RuntimeError`

2. **之前的错误修复方式:**
   ```python
   # ❌ 错误：在背景线程中访问 current_app
   def run_orchestrated_diagnosis():  # 背景线程
       from flask import current_app
       app_instance = current_app._get_current_object()  # 这里会出错！
   ```

3. **正确的修复方式:**
   ```python
   # ✅ 正确：在主线程中捕获，通过闭包传递
   # 主线程
   app_instance = flask_current_app._get_current_object()
   
   def run_orchestrated_diagnosis():  # 背景线程
       def run_with_app_context():
           with app_instance.app_context():  # 使用捕获的实例
               ...
   ```
