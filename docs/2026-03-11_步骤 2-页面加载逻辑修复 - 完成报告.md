## 一、问题发现

### 1.1 症状

页面显示：
- 报告摘要：0 总样本数、0 关键词、0 涉及品牌
- 品牌分布：空
- 情感分布：空
- 关键词云：空

### 1.2 根因分析

**问题代码** (`report-v2.js` 第 656-670 行):
```javascript
// ❌ 错误的 Storage key
const storageData = wx.getStorageSync('last_diagnostic_results');

if (storageData && (storageData.executionId === id || storageData.version === '2.0')) {
  report = {
    brandDistribution: storageData.dashboard?.brandDistribution || {},
    // ...
  };
}
```

**问题点**:
1. Storage key 错误：使用 `'last_diagnostic_results'` 而不是 `'diagnosis_result_' + id`
2. 数据路径错误：使用 `storageData.dashboard?.brandDistribution` 而不是 `storageData.data?.brandDistribution`
3. 导致 Storage 中有数据但读取不到

### 1.3 数据流对比

**实际保存的 key** (`utils/storage-manager.js`):
```javascript
const key = StorageKey.DIAGNOSIS_RESULT + executionId;
// 结果：'diagnosis_result_xxx'
```

**页面读取的 key** (修复前):
```javascript
wx.getStorageSync('last_diagnostic_results');
// ❌ key 不匹配！
```

**页面读取的 key** (修复后):
```javascript
const storageKey = 'diagnosis_result_' + id;
wx.getStorageSync(storageKey);
// ✅ key 匹配！
```

## 二、修复方案

### 2.1 修复内容

**修复文件**: `miniprogram/pages/report-v2/report-v2.js`

**修复位置**: `loadReportData()` 方法，第 656-688 行

**修复前**:
```javascript
// Step 3: 如果云函数也没有数据，从 Storage 读取备份
if (!report || !report.brandDistribution || Object.keys(report.brandDistribution).length === 0) {
  console.log('[ReportPageV2] 云函数无数据，尝试从 Storage 读取备份...');
  const storageData = wx.getStorageSync('last_diagnostic_results');

  if (storageData && (storageData.executionId === id || storageData.version === '2.0')) {
    console.log('[ReportPageV2] ✅ 从 Storage 获取报告数据');
    report = {
      brandDistribution: storageData.dashboard?.brandDistribution || {},
      sentimentDistribution: storageData.dashboard?.sentimentDistribution || {},
      keywords: storageData.dashboard?.keywords || [],
      brandScores: storageData.brandScores || {},
      status: 'completed',
      progress: 100,
      stage: 'completed'
    };
    dataSource = 'storage';
  }
}
```

**修复后**:
```javascript
// Step 3: 如果云函数也没有数据，从 Storage 读取备份
if (!report || !report.brandDistribution || Object.keys(report.brandDistribution).length === 0) {
  console.log('[ReportPageV2] 📥 步骤 3: 云函数无数据，尝试从 Storage 读取备份...');
  
  // 【P0 关键修复 - 2026-03-11】修复 Storage key，使用 diagnosis_result_ + executionId
  const storageKey = 'diagnosis_result_' + id;
  console.log('[ReportPageV2] Storage key:', storageKey);
  
  const storageData = wx.getStorageSync(storageKey);
  console.log('[ReportPageV2] Storage 原始数据:', storageData);
  console.log('[ReportPageV2] Storage data 字段:', storageData?.data);

  if (storageData && storageData.executionId === id) {
    console.log('[ReportPageV2] ✅ 从 Storage 获取报告数据');
    console.log('[ReportPageV2] storageData.data.brandDistribution:', storageData.data?.brandDistribution);
    
    report = {
      brandDistribution: storageData.data?.brandDistribution || {},  // 修复数据路径
      sentimentDistribution: storageData.data?.sentimentDistribution || {},
      keywords: storageData.data?.keywords || [],
      brandScores: storageData.data?.brandScores || {},
      status: 'completed',
      progress: 100,
      stage: 'completed'
    };
    dataSource = 'storage';
    console.log('[ReportPageV2] Storage 解析数据:', report);
  } else {
    console.log('[ReportPageV2] ⚠️ Storage 无数据或 executionId 不匹配');
  }
}
```

### 2.2 关键修复点

| 修复点 | 修复前 | 修复后 |
|-------|--------|--------|
| **Storage key** | `'last_diagnostic_results'` | `'diagnosis_result_' + id` |
| **数据路径** | `storageData.dashboard?.brandDistribution` | `storageData.data?.brandDistribution` |
| **验证条件** | `storageData.executionId === id \|\| storageData.version === '2.0'` | `storageData && storageData.executionId === id` |
| **调试日志** | 简单 | 详细（包含 key、原始数据、解析数据） |
