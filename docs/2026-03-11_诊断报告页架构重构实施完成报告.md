## 一、架构重构概述

### 1.1 核心原则

**数据独立**: 每层数据独立获取，不依赖其他层

**异步处理**: 使用 setTimeout 异步处理，不阻塞主线程

**容错设计**: 某层失败不影响其他层

**渐进增强**: 先显示核心，再逐步增强

### 1.2 分层设计

| 层级 | 内容 | 时机 | 数据来源 | 数据量 |
|-----|------|------|---------|--------|
| **第 1 层** | 核心信息 | 0ms | 直接获取 | <1KB |
| **第 2 层** | 分析数据 | 100ms | 后端预计算 | ~5KB |
| **第 3 层** | 详细结果 | 200ms | 异步简化 | ~5KB |

### 2.2 第 2 层：分析数据（100ms）

**实施代码**:
```javascript
setTimeout(() => {
  try {
    // 后端预计算的评分维度
    const dimensionScores = results.dimension_scores || results.dimensionScores || {};
    
    // 后端预计算的核心指标
    const exposureAnalysis = results.exposure_analysis || {};
    const brandData = exposureAnalysis.brand_details?.[mainBrand] || {};
    
    // 后端预计算的问题诊断
    const recommendationData = results.recommendation_data || {};
    const riskLevels = recommendationData.risk_levels || {};
    
    // 后端预计算的信源分析
    const sourceIntelligence = results.source_intelligence || {};
    
    this.setData({
      // 评分维度（后端预计算）
      overallAuthority: dimensionScores.authority || 0,
      overallVisibility: dimensionScores.visibility || 0,
      overallPurity: dimensionScores.purity || 0,
      overallConsistency: dimensionScores.consistency || 0,
      
      // 核心指标（后端预计算）
      sovShare: brandData.sov_share ? Math.round(brandData.sov_share * 100) : 0,
      sentimentScore: brandData.sentiment_score || 0,
      physicalRank: brandData.rank || 0,
      influenceScore: Math.round((
        dimensionScores.authority + dimensionScores.visibility +
        dimensionScores.purity + dimensionScores.consistency
      ) / 4),
      hasMetrics: true,
      
      // 问题诊断（后端预计算）
      highRisks: riskLevels.high || [],
      mediumRisks: riskLevels.medium || [],
      suggestions: recommendationData.priority_recommendations || [],
      
      // 信源分析（后端预计算）
      sourceList: sourceIntelligence.source_pool?.slice(0, 10) || []
    });
  } catch (error) {
    // 容错：设置默认值，不影响其他层
    this.setData({
      overallAuthority: 0, overallVisibility: 0,
      overallPurity: 0, overallConsistency: 0,
      hasMetrics: false, highRisks: [],
      mediumRisks: [], suggestions: [], sourceList: []
    });
  }
}, 100);
```

**字段**:
- ✅ 评分维度（后端预计算）
- ✅ 核心指标（后端预计算）
- ✅ 问题诊断（后端预计算）
- ✅ 信源分析（后端预计算）

**特点**:
- ✅ 使用后端预计算数据
- ✅ 不依赖第 3 层
- ✅ 独立获取
- ✅ 容错处理

### 2.3 第 3 层：详细结果（200ms）

```javascript
setTimeout(() => {
  try {
    const detailedResults = results.detailed_results || results.detailedResults || [];
    
    // 异步简化数据（只处理前 5 条）
    const simplifiedResults = detailedResults.slice(0, 5).map((r, index) => ({
      id: r.id || `result_${index}`,
      brand: r.brand || r.brandName,
      model: r.aiModel || r.ai_model || '未知模型',
      score: r.score,
      question: (r.question || '').substring(0, 50),
      response: (r.response || r.answer || '').substring(0, 100),
      truncated: (r.response?.length || r.answer?.length || 0) > 100,
      fullResponse: r.response || r.answer || '',
      expanded: false
    }));
    
    // 保存完整数据到全局变量
    const app = getApp();
    app.globalData.currentReportData = {
      detailedResults: detailedResults,
      executionId: record.executionId
    };
    
    this.setData({
      detailedResults: simplifiedResults,
      aiModels: results.ai_models || [],
      competitors: results.competitors || [],
      questions: results.questions || [],
      hasMoreResults: detailedResults.length > 5,
      totalResults: detailedResults.length
    });
  } catch (error) {
    // 容错：设置默认值
    this.setData({
      detailedResults: [], aiModels: [],
      competitors: [], questions: [],
      hasMoreResults: false, totalResults: 0
    });
  }
}, 200);
```

**字段**:
- ✅ detailedResults（简化，前 5 条）
- ✅ aiModels（直接获取）
- ✅ competitors（直接获取）
- ✅ questions（直接获取）
- ✅ hasMoreResults（计算）
- ✅ totalResults（计算）

**特点**:
- ✅ 异步处理
- ✅ 文本截断
- ✅ 分页加载
- ✅ 容错处理

## 三、关键改进

### 3.1 数据依赖消除

**改进前** ❌:
```javascript
// 错误：第 1 层等待第 2 层计算
const dimensionScores = this.calculateDimensionScores(detailedResults);
this.setData({ dimensionScores }); // 阻塞！

// 错误：第 2 层等待第 3 层处理
const simplifiedResults = this.simplifyResults(detailedResults);
this.setData({ simplifiedResults }); // 阻塞！
```

**改进后** ✅:
```javascript
// 正确：直接使用后端预计算数据
const dimensionScores = results.dimension_scores || {};
this.setData({
  overallAuthority: dimensionScores.authority || 0
}); // 不阻塞！

// 正确：异步处理，不阻塞
setTimeout(() => {
  const simplifiedResults = detailedResults.slice(0, 5);
  this.setData({ detailedResults: simplifiedResults });
}, 200);
```

### 3.2 容错设计

**每层独立 try-catch**:
```javascript
// 第 1 层
try {
  this.setData({ ... });
} catch (error) {
  // 不影响其他层
}

// 第 2 层
setTimeout(() => {
  try {
    this.setData({ ... });
  } catch (error) {
    // 不影响其他层
  }
}, 100);

// 第 3 层
setTimeout(() => {
  try {
    this.setData({ ... });
  } catch (error) {
    // 不影响其他层
  }
}, 200);
```
