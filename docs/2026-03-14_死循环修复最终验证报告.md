## 📊 修复效果预测

### 正常场景

```
用户点击历史记录
    ↓
检查 _isLoadingHistory (false)
    ↓
设置 _isLoadingHistory = true
    ↓
调用 reportService.getFullReport()
    ↓
API 成功返回
    ↓
设置 _isLoadingHistory = false
    ↓
跳转到报告详情页
    ↓
✅ 页面正常显示
```

### API 失败场景

```
用户点击历史记录
    ↓
检查 _isLoadingHistory (false)
    ↓
设置 _isLoadingHistory = true
    ↓
调用 reportService.getFullReport()
    ↓
API 失败
    ↓
重试 0/3 → 等待 1s → 重试 1/3
    ↓
重试 1/3 → 等待 2s → 重试 2/3
    ↓
重试 2/3 → 等待 3s → 达到最大重试
    ↓
返回错误报告
    ↓
设置 _isLoadingHistory = false
    ↓
显示错误对话框
    ↓
用户点击"重试"
    ↓
currentPage.onLoad() (非递归)
    ↓
✅ 页面重新加载
```

### 快速点击场景

```
用户快速点击 3 次
    ↓
第 1 次：_isLoadingHistory = false → 执行加载
    ↓
第 2 次：_isLoadingHistory = true → 跳过
    ↓
第 3 次：_isLoadingHistory = true → 跳过
    ↓
控制台：⚠️ 正在加载中，跳过重复请求
    ↓
✅ 只响应第一次点击
```
