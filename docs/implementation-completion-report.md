### 2.3 收藏功能数据流

```
用户在列表页点击收藏
    ↓
添加到 wx.setStorageSync('favorites')
    ↓
收藏列表实时更新
    ↓
详情页加载时检查收藏状态
    ↓
显示已收藏/未收藏状态
    ↓
用户可随时取消收藏
```

**收藏数据结构**:
```javascript
{
  executionId: "xxx",      // 执行 ID
  brandName: "品牌名称",    // 品牌名称
  favoritedAt: 1234567890, // 收藏时间戳
  timestamp: 1234567890,   // 时间戳（兼容字段）
  overallScore: 85         // 综合评分（可选）
}
```

## 五、技术亮点

### 5.1 状态管理设计

**三级缓存机制**:
```
内存缓存 (app.globalData.pageStateCache)
    ↓
Storage 持久化 (page_state_xxx)
    ↓
24 小时自动过期
```

**优点**:
- 快速响应（内存优先）
- 持久化保存（Storage 备份）
- 自动清理（过期机制）

### 5.2 导航服务统一

**统一入口**:
```javascript
navigationService.navigateToXxx()
```

**优点**:
- 代码复用
- 易于维护
- 错误处理统一

### 5.3 收藏功能设计

**数据同步**:
- 列表页和详情页共享收藏数据
- 实时同步收藏状态
- 支持跨页面收藏管理
