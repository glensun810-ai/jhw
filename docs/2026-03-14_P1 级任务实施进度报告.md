## 📝 技术说明

### 问题模板库实现细节

**数据结构**:
```json
{
  "id": "tpl_001",
  "name": "品牌知名度诊断",
  "category": "品牌分析",
  "industry": "通用",
  "description": "评估品牌在目标市场中的知名度和认知度",
  "questions": [
    "请分析 {brand} 在消费者心中的品牌知名度如何？",
    "{brand} 的主要品牌优势是什么？",
    "消费者提到 {brand} 时最先想到的三个关键词是什么？"
  ],
  "tags": ["知名度", "认知度", "品牌形象"],
  "usage_count": 0,
  "is_default": true
}
```

**品牌占位符替换**:
```javascript
// 前端调用
const questions = questionTemplateService.replaceBrandPlaceholder(
  template.questions,
  '德施曼'
);
// 输出：["请分析 德施曼 在消费者心中的品牌知名度如何？", ...]
```
