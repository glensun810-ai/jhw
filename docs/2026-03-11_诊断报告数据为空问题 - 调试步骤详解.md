   ```python
   # backend_python/wechat_backend/services/diagnosis_orchestrator.py
   # 搜索：brand_distribution, brandScores
   
   def _generate_brand_distribution(self, results):
       """生成品牌分布数据"""
       print(f'[诊断编排器] 生成品牌分布，results 数量：{len(results)}')
       
       # 检查是否有数据
       if not results:
           print('[诊断编排器] ❌ results 为空！')
           return {'total_count': 0, 'data': {}}
       
       # 统计品牌出现次数
       brand_counts = {}
       for result in results:
           brand = result.get('brand', 'Unknown')
           brand_counts[brand] = brand_counts.get(brand, 0) + 1
       
       print(f'[诊断编排器] 品牌统计：{brand_counts}')
       
       total = sum(brand_counts.values())
       distribution = {
           'total_count': total,
           'data': {
               brand: {
                   'count': count,
                   'percentage': round(count / total * 100, 2)
               }
               for brand, count in brand_counts.items()
           }
       }
       
       print(f'[诊断编排器] 品牌分布：{distribution}')
       return distribution
   ```

## 四、常见问题及解决方案

### 问题 A: Storage 中无数据

**症状**:
```javascript
console.log('诊断 result keys:', []);  // 空数组
```

**可能原因**:
1. 诊断完成后未调用 `saveDiagnosisResult`
2. executionId 不匹配
3. storage 保存失败

**解决方案**:
```javascript
// 1. 检查 pages/index/index.js 的 handleDiagnosisComplete
const saveSuccess = saveDiagnosisResult(executionId, {
  brandName: this.data.brandName,
  // ... 确保传递了 brandDistribution 等数据
  rawResponse: parsedStatus  // 保存完整响应
});

console.log('Storage 保存结果:', saveSuccess);

// 2. 检查 executionId 是否正确
console.log('executionId:', executionId);
```

---

### 问题 B: 数据结构和不匹配

**症状**:
```javascript
// 页面期望
brandDistribution: {total_count: 10, data: {...}}

// 实际保存
brandScores: {...}  // 字段名不匹配
```

**解决方案**:
```javascript
// 修改解析逻辑
_parseReportData: function(rawData) {
  // 支持多种字段名
  const brandDistribution = 
    rawData.data?.brandDistribution || 
    rawData.data?.brandScores || 
    rawData.brandDistribution || 
    rawData.brandScores || 
    {};
  
  console.log('解析后的 brandDistribution:', brandDistribution);
  
  return {
    brandDistribution,
    // ...
  };
}
```
