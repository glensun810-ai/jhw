## 四、组件使用说明

### 4.1 评分仪表盘

```xml
<score-gauge 
  score="{{overallScore}}"
  grade="{{overallGrade}}"
  sov-share="{{sovShare}}"
  sentiment-score="{{sentimentScore}}"
  physical-rank="{{physicalRank}}"
  influence-score="{{influenceScore}}"
/>
```

### 4.2 问题诊断墙

```xml
<question-wall 
  title="问题诊断"
  high-risks="{{highRisks}}"
  medium-risks="{{mediumRisks}}"
  suggestions="{{suggestions}}"
/>
```

### 4.3 信源列表

```xml
<source-list 
  title="信源分析"
  sources="{{sourceList}}"
/>
```

## 五、数据格式要求

### 5.1 核心指标数据

```javascript
{
  sovShare: 35,           // 声量份额 (%)
  sentimentScore: 82,     // 情感得分 (0-100)
  physicalRank: 2,        // 物理排名
  influenceScore: 75      // 影响力评分 (0-100)
}
```

### 5.2 问题诊断数据

```javascript
{
  highRisks: [
    { id: '1', text: '负面评价较多', source: '知乎' }
  ],
  mediumRisks: [
    { id: '2', text: '曝光度不足', source: '小红书' }
  ],
  suggestions: [
    { id: '1', text: '加大内容投放', priority: '高' }
  ]
}
```

### 5.3 信源数据

```javascript
{
  sourceList: [
    {
      id: 'src_001',
      site_name: '知乎',
      url: 'https://...',
      citation_count: 15,
      domain_authority: '高',
      risk_level: 'Low'
    }
  ]
}
```
