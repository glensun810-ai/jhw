## 二、详细修复内容

### 2.1 后端修复

#### 修复 1: 完善降级方案 (`diagnosis_report_service.py`)

**新增方法**:

1. **`_create_fallback_report()`** - 创建降级报告
   - 处理 4 种错误场景：not_found, failed, timeout, no_results
   - 提供友好的错误建议
   - 包含质量提示信息

2. **`_create_partial_fallback_report()`** - 处理部分结果
   - 诊断进行中的状态展示
   - 进度百分比显示
   - 稍后重试建议

3. **`_ensure_complete_analysis()`** - 确保分析数据完整
   - 缺失分析类型提供默认值
   - 记录缺失警告

4. **`_check_data_quality()`** - 检查数据质量
   - 低质量结果检测
   - 情感分布异常检测
   - 分析数据完整性检查

**错误场景处理**:

| 场景 | 触发条件 | 处理方式 |
|------|---------|---------|
| 报告不存在 | `report == null` | 返回 not_found 降级报告 |
| 诊断失败 | `status == 'failed'` | 返回 failed 降级报告 |
| 处理超时 | `status == 'processing'` 且 > 10 分钟 | 返回 timeout 降级报告 |
| 结果为空 | `results.length == 0` | 根据进度返回部分或 no_results 报告 |
| 分析缺失 | 分析数据为空 | 提供默认值并记录警告 |
| 质量低 | `quality_score < 60` | 添加质量警告 |

**降级报告结构**:

```json
{
  "report": {...},
  "results": [],
  "analysis": {},
  "brandDistribution": {"data": {}, "total_count": 0},
  "sentimentDistribution": {"data": {}, "total_count": 0},
  "keywords": [],
  "error": {
    "status": "not_found|failed|timeout|no_results",
    "message": "错误信息",
    "suggestion": "用户建议"
  },
  "validation": {
    "is_valid": false,
    "errors": ["错误列表"],
    "warnings": []
  },
  "qualityHints": {
    "has_low_quality_results": false,
    "has_partial_analysis": true,
    "warnings": []
  }
}
```

#### 修复 2: 增强验证服务 (`diagnosis_report_service.py`)

**新增验证项**:

1. **报告主数据验证** (`_validate_report_data`)
   - 必填字段检查
   - 状态字段有效性
   - 进度值范围验证

2. **结果数据验证** (`_validate_results_data`)
   - 结果数组格式
   - 每个结果的必填字段
   - 结果内容检查
   - 质量评分验证

3. **分析数据验证** (`_validate_analysis_data`)
   - 必需分析类型检查
   - 可选分析类型提示
   - 分析数据完整性

4. **聚合数据验证** (`_validate_aggregation_data`)
   - 品牌分布格式
   - 情感分布完整性
   - 关键词数组验证

5. **校验和验证** (`_validate_checksum`)
   - 校验和存在性
   - 校验和匹配性

**质量评分算法**:

```
基础分：100 分

扣分项:
- 每个错误：-20 分
- 每个警告：-5 分
- 每个质量问题：-3 分
- 报告验证失败：-10 分
- 结果验证失败：-10 分
- 分析验证失败：-5 分
- 聚合验证失败：-5 分
- 校验和验证失败：-10 分

最终分数：max(0, min(100, 计算分数))
```

**验证返回结构**:

```json
{
  "is_valid": true,
  "errors": [],
  "warnings": [],
  "quality_issues": [],
  "quality_score": 95,
  "details": {
    "report_valid": true,
    "results_valid": true,
    "analysis_valid": true,
    "aggregation_valid": true,
    "checksum_valid": true
  }
}
```

**错误处理增强**:

```javascript
async getFullReport(executionId) {
    try {
        const report = await wx.cloud.callFunction({...});
        
        // 处理空报告
        if (!report) {
            return this._createEmptyReportWithSuggestion(
                '报告不存在', 
                'not_found'
            );
        }
        
        // 检查验证信息
        if (validation && !validation.is_valid) {
            console.warn('Report validation failed:', validation.errors);
        }
        
        // 检查质量评分
        if (qualityScore < 60) {
            console.warn('Low quality report:', qualityScore);
            processedReport.lowQualityWarning = true;
        }
        
        return processedReport;
    } catch (error) {
        // 错误时返回空报告
        return this._createEmptyReportWithSuggestion(
            '获取报告失败', 
            'error'
        );
    }
}
```
