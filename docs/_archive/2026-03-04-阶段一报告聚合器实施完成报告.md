## 二、功能详解

### 2.1 品牌分数计算

**计算逻辑**:
```python
# 计算平均分
total_score = sum(r.get('score', 50) for r in brand_results)
avg_score = round(total_score / len(brand_results))

# 计算等级
grade = self._get_grade(avg_score)  # A+/A/B/C/D

# 计算各项分数
scores[brand] = {
    'overallScore': avg_score,
    'overallGrade': grade,
    'overallAuthority': round(avg_score * 0.9),    # 权威性
    'overallVisibility': round(avg_score * 0.85),  # 可见性
    'overallPurity': round(avg_score * 0.9),       # 纯净度
    'overallConsistency': round(avg_score * 0.8),  # 一致性
    'overallSummary': self._get_score_summary(avg_score)
}
```

**等级映射**:
| 分数范围 | 等级 |
|---------|------|
| ≥90 | A+ |
| ≥80 | A |
| ≥70 | B |
| ≥60 | C |
| <60 | D |

---

### 2.2 SOV (Share of Voice) 计算

**计算公式**:
```python
brand_sov = (brand_mentions / total_mentions) * 100
competitor_sov = (competitor_mentions / total_mentions) * 100
```

**示例**:
- 主品牌提及：3 次
- 竞品提及：2 次
- 总提及：5 次
- 主品牌 SOV: 60%
- 竞品 SOV: 40%

risk_score = (negative_count / total) * 100
