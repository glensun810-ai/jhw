## 一、问题描述

### 错误信息

```
Error: can not find module : , require args is ../../utils/request
    at analytics.js? [sm]:10
```

### 问题原因

`api/analytics.js` 中引用路径错误：
```javascript
// 错误 - 从 api/目录到 utils/应该是 ../utils
const { get } = require('../../utils/request');

// 正确
const { get } = require('../utils/request');
```

---

## 二、修复方案

### 修复文件

| 文件 | 变更 | 状态 |
|-----|------|------|
| `api/analytics.js` | 修正 require 路径 | ✅ |

### 修复内容

```javascript
// 修复前
const { get } = require('../../utils/request');
const { API_ENDPOINTS } = require('../../utils/config');

// 修复后
const { get } = require('../utils/request');
const { API_ENDPOINTS } = require('../utils/config');
```
