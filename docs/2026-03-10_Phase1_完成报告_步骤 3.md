## 二、新增组件详情

### 2.1 趋势图组件 (line-chart)

**功能**: 展示声量份额、情感得分等随时间变化的趋势

**属性**:
```javascript
{
  data: [{value: 85, label: '1 月'}, {value: 72, label: '2 月'}],
  title: '声量份额趋势',
  color: '#4A7BFF',
  showValue: true,
  showYAxis: true
}
```

**使用示例**:
```xml
<line-chart 
  data="{{trendData}}"
  title="声量份额趋势"
  color="#4A7BFF"
/>
```

**技术特点**:
- 自动计算 Y 轴最大值
- 支持自定义颜色
- 数据点显示数值
- X 轴标签自动排列

### 2.2 雷达图组件 (radar-chart)

**功能**: 展示 AI 平台认知差异等多维度对比

**属性**:
```javascript
{
  data: [
    {name: '声量', value: 85},
    {name: '情感', value: 72},
    {name: '权威', value: 65},
    {name: '可见', value: 78},
    {name: '纯净', value: 82},
    {name: '一致', value: 70}
  ],
  title: 'AI 平台认知差异',
  color: '#4A7BFF',
  maxValue: 100,
  showDataList: true
}
```

**使用示例**:
```xml
<radar-chart 
  data="{{platformData}}"
  title="AI 平台认知差异"
  color="#4A7BFF"
/>
```

**技术特点**:
- 自动计算数据点位置
- 支持任意维度（3-8 维）
- 背景网格自动绘制
- 数据列表可选显示

### 2.3 词云组件 (word-cloud)

**功能**: 展示高频问题 TOP10 等

**属性**:
```javascript
{
  data: [
    {name: '品牌知名度如何', value: 156},
    {name: '产品质量怎么样', value: 132},
    {name: '和竞品比哪个更好', value: 98}
  ],
  title: '高频问题',
  subtitle: 'TOP10',
  showCount: true,
  showList: true,
  maxTags: 10
}
```

**使用示例**:
```xml
<word-cloud 
  data="{{questionData}}"
  title="高频问题"
  subtitle="TOP10"
  showList="{{true}}"
/>
```

**技术特点**:
- 词云 + 列表双模式
- 自动计算字体大小
- 前 3 名高亮显示
- 颜色自动分配

## 三、组件使用指南

### 3.1 在页面中注册组件

```json
{
  "usingComponents": {
    "line-chart": "../../components/charts/line-chart/line-chart",
    "radar-chart": "../../components/charts/radar-chart/radar-chart",
    "word-cloud": "../../components/charts/word-cloud/word-cloud",
    "bar-chart": "../../components/charts/bar-chart/bar-chart",
    "question-wall": "../../components/report/question-wall/question-wall",
    "source-list": "../../components/report/source-list/source-list"
  }
}
```

### 3.2 在页面中使用组件

```xml
<!-- 趋势图 -->
<line-chart 
  data="{{trendData}}"
  title="声量份额趋势"
/>

<!-- 雷达图 -->
<radar-chart 
  data="{{platformData}}"
  title="AI 平台认知差异"
/>

<!-- 词云 -->
<word-cloud 
  data="{{questionData}}"
  title="高频问题"
  subtitle="TOP10"
/>
```

## 四、统计分析页面集成

### 4.1 页面结构建议

```xml
<view class="analytics-page">
  <!-- Tab 切换 -->
  <view class="tabs">
    <view class="tab active">品牌对比</view>
    <view class="tab">趋势分析</view>
    <view class="tab">AI 平台</view>
    <view class="tab">问题聚合</view>
  </view>
  
  <!-- 品牌对比 -->
  <view wx:if="{{currentTab === 'brand'}}">
    <bar-chart data="{{brandData}}" title="品牌综合评分对比" />
  </view>
  
  <!-- 趋势分析 -->
  <view wx:if="{{currentTab === 'trend'}}">
    <line-chart data="{{trendData}}" title="声量份额趋势" />
  </view>
  
  <!-- AI 平台对比 -->
  <view wx:if="{{currentTab === 'platform'}}">
    <radar-chart data="{{platformData}}" title="AI 平台认知差异" />
  </view>
  
  <!-- 问题聚合 -->
  <view wx:if="{{currentTab === 'question'}}">
    <word-cloud data="{{questionData}}" title="高频问题 TOP10" />
  </view>
</view>
```

### 4.2 后端 API 需求

| API | 功能 | 数据格式 |
|-----|------|---------|
| `/api/analytics/brand-compare` | 品牌对比 | `[{name: '品牌 A', value: 85}]` |
| `/api/analytics/trend` | 趋势分析 | `[{value: 35, label: '1 月'}]` |
| `/api/analytics/platform-compare` | AI 平台对比 | `[{name: '声量', value: 85}]` |
