## 🎨 核心视觉元素对比

### 1. 页面背景
**优化前**:
```css
background: linear-gradient(135deg, #f5f7fa 0%, #c3cfe2 100%);
```

**优化后**:
```css
background: linear-gradient(135deg, #1a1f29 0%, #121826 100%);
```

### 2. 卡片样式
**优化前**:
```css
background: #ffffff;
box-shadow: 0 8rpx 24rpx rgba(0, 0, 0, 0.08);
```

**优化后**:
```css
background: #1A202C;
border: 1rpx solid #303030;
box-shadow: 0 8rpx 24rpx rgba(0, 0, 0, 0.3);
```

### 3. 按钮样式
**优化前** (紫色主题):
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
color: #ffffff;
```

**优化后** (科技青主题):
```css
background: linear-gradient(135deg, #00E5FF 0%, #00F5A0 100%);
color: #121826;
```

### 4. 分数显示
**优化前**:
```css
color: #667eea; /* 紫色 */
```

**优化后**:
```css
color: #00E5FF; /* 科技青 */
font-family: 'Courier New', 'DIN', monospace; /* 数字字体 */
```

## 📝 使用说明

### 开发时注意事项

1. **新增页面时**，务必导入全局样式：
   ```css
   @import "../../app.wxss";
   ```

2. **使用主题色时**，使用统一色值：
   ```css
   --primary-color: #121826;
   --accent-color: #00E5FF;
   --success-color: #00F5A0;
   ```

3. **卡片样式统一**：
   ```css
   background: #1A202C;
   border: 1rpx solid #303030;
   box-shadow: 0 8rpx 24rpx rgba(0, 0, 0, 0.3);
   ```

4. **数字显示使用专用字体**：
   ```css
   font-family: 'Courier New', 'DIN', monospace;
   ```
