## 二、品牌维度分析

### 2.1 功能特性

**分析指标**:
- ✅ 综合评分对比（柱状图）
- ✅ 声量份额对比（饼图/堆叠柱状图）
- ✅ 情感得分对比（进度条）
- ✅ 物理排名对比（排名表）

**UI 组件**:
- 指标切换选择器
- 4 个分析区块
- 可视化图表

### 2.2 技术实现

**页面路径**: `pages/analytics/brand-dimension/`

**核心方法**:
```javascript
loadScoreCompare()   // 加载综合评分对比
loadSOVData()        // 加载声量份额数据
loadSentimentData()  // 加载情感得分数据
loadRankingData()    // 加载物理排名数据
```

**数据可视化**:
- 柱状图：综合评分
- 堆叠图：声量份额
- 进度条：情感得分
- 排名表：物理排名

## 三、时间维度分析

### 3.1 功能特性

**分析指标**:
- ✅ 评分趋势（折线图）
- ✅ 声量趋势（柱状图）
- ✅ 情感趋势（散点图）
- ✅ 排名趋势（折线图）

**UI 组件**:
- 品牌选择器
- 时间范围切换
- 4 个趋势图表

### 3.2 技术实现

**页面路径**: `pages/analytics/time-dimension/`

**核心方法**:
```javascript
loadScoreTrend()      // 加载评分趋势
loadVolumeData()      // 加载声量数据
loadSentimentTrend()  // 加载情感趋势
loadRankingTrend()    // 加载排名趋势
```

**交互功能**:
- 品牌筛选
- 时间范围切换（7 天/30 天/90 天）
- 下拉刷新
