## 六、组件使用示例

### 6.1 在 JSON 中注册组件

```json
{
  "usingComponents": {
    "question-wall": "../../components/report/question-wall/question-wall",
    "source-list": "../../components/report/source-list/source-list"
  }
}
```

### 6.2 在 WXML 中使用

```xml
<question-wall 
  title="问题诊断"
  high-risks="{{highRisks}}"
  medium-risks="{{mediumRisks}}"
  suggestions="{{suggestions}}"
/>

<source-list 
  title="信源分析"
  sources="{{sourceList}}"
/>
```
