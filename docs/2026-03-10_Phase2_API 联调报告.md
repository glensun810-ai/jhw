## 五、混合模式设计

### 5.1 数据优先级

```
API 数据 (优先)
  ↓ (API 失败)
本地数据 (降级)
```

### 5.2 实现逻辑

```javascript
async loadBrandCompare() {
  try {
    // 尝试 API 数据
    const res = await analyticsApi.getBrandCompareApi();
    this.setData({
      brandCompareData: res.data.brands,
      useApi: true
    });
  } catch (error) {
    // 降级到本地数据
    console.warn('API 失败，使用本地数据');
    this.loadBrandCompareLocal();
  }
}
```

## 六、性能优化

### 6.1 缓存策略

**API 数据缓存**:
- 缓存时间：5 分钟
- 缓存 key: `analytics_{type}_{timeRange}`

**实现**:
```javascript
const cacheKey = `analytics_brand_${this.data.timeRange}`;
const cached = wx.getStorageSync(cacheKey);

if (cached && Date.now() - cached.timestamp < 5 * 60 * 1000) {
  // 使用缓存数据
  this.setData({ brandCompareData: cached.data });
} else {
  // 请求新数据
  const res = await analyticsApi.getBrandCompareApi();
  wx.setStorageSync(cacheKey, {
    data: res.data.brands,
    timestamp: Date.now()
  });
}
```

### 6.2 请求优化

**防重复请求**:
```javascript
if (this.data.brandLoaded) {
  return; // 已加载，跳过
}
```

**批量请求**:
```javascript
// 同时加载多个维度
Promise.all([
  this.loadBrandCompare(),
  this.loadTrendAnalysis()
]);
```
