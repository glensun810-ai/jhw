## 一、问题描述

### 1.1 用户反馈

报告详情页显示浅蓝色背景，与首页深色科技主题不一致：
- 品牌分布组件：白色背景 `#ffffff`
- 情感分布组件：白色背景 `#ffffff`
- 关键词云组件：白色背景 `#ffffff`
- 切换按钮：浅蓝色背景 `#f0f5ff`

### 1.2 问题截图

```

│  0 总样本数    0 关键词    0 涉及品牌  │

│                                 │
│           暂无数据               │

```

## 三、修复方案

### 3.1 修复策略

**统一深色科技主题**：
1. 组件背景改为透明，显示页面深色背景
2. 文字颜色改为浅色 `#e8e8e8`
3. 边框改为半透明白色 `rgba(255, 255, 255, 0.1)`
4. 按钮和交互元素使用科技青 `#00E5FF`

### 3.2 修复内容

#### 修复 1: 品牌分布组件 ✅

**文件**: `miniprogram/components/brand-distribution/brand-distribution.wxss`

**关键修复**:
```css

.brand-distribution {
  background: transparent;  /* 透明背景 */
}

.title {
  color: #e8e8e8;  /* 浅色文字 */
}

.toggle-btn {
  color: #00E5FF;
  background-color: rgba(0, 229, 255, 0.1);
  border: 1rpx solid rgba(0, 229, 255, 0.3);
}

.legend-item {
  background-color: rgba(255, 255, 255, 0.05);
  border: 1rpx solid rgba(255, 255, 255, 0.1);
}

.bar-wrapper {
  background-color: rgba(255, 255, 255, 0.1);
}
```

#### 修复 2: 情感分布组件 ✅
