## 二、统计分析页面详情

### 2.1 页面结构

```
统计分析页面
├── 页面头部
│   ├── 标题
│   └── 副标题
├── 时间范围选择器
│   ├── 全部
│   ├── 近 7 天
│   ├── 近 30 天
│   └── 近 90 天
├── Tab 切换
│   ├── 品牌对比 📊
│   ├── 趋势分析 📈
│   ├── AI 平台 🤖
│   └── 问题聚合 ❓
└── 内容区域
    ├── 品牌对比（柱状图）
    ├── 趋势分析（趋势图）
    ├── AI 平台（雷达图）
    └── 问题聚合（词云）
```

### 2.2 功能特性

**品牌对比**:
- 柱状图展示各品牌综合评分
- 按平均分排序
- 自动聚合历史数据

**趋势分析**:
- 趋势图展示声量份额变化
- 最近 10 次诊断记录
- 按时间顺序排列

**AI 平台对比**:
- 雷达图展示各平台认知差异
- 多维度对比（声量、情感等）
- 取前 6 个平台

**问题聚合**:
- 词云 + 列表双模式
- TOP10 高频问题
- 前 3 名高亮显示

### 2.3 数据来源

**本地存储**:
```javascript
// 从 diagnosis_history_list 读取
wx.getStorageSync('diagnosis_history_list')

// 数据结构
[
  {
    executionId: "xxx",
    brandName: "品牌 A",
    overallScore: 85,
    aiModels: [{name: "doubao"}],
    questions: ["问题 1"],
    createdAt: "2026-03-10"
  }
]
```

**数据处理**:
- 品牌对比：按品牌聚合，计算平均分
- 趋势分析：按时间排序，取最近 10 条
- AI 平台：按平台聚合，计算平均分
- 问题聚合：按问题聚合，统计频次

## 三、组件使用示例

### 3.1 柱状图

```xml
<bar-chart 
  data="{{brandCompareData}}"
  title="品牌综合评分对比"
  color="#4A7BFF"
/>
```

```javascript

  data: {
    brandCompareData: [
      {name: '品牌 A', value: 85},
      {name: '品牌 B', value: 72},
      {name: '品牌 C', value: 68}
    ]
  }
})
```

### 3.2 趋势图

```xml
<line-chart 
  data="{{trendData}}"
  title="声量份额趋势"
  color="#00F5A0"
```
