### P1-04 | 优化关键词云生成质量 ✅

**状态**: 已完成（新增实现）

**实施内容**:
1. 实现 TF-IDF 关键词提取
2. 实现 TextRank 关键词提取
3. 实现混合提取方法
4. 中文分词优化（jieba）
5. 停用词过滤

**修改文件**:
- `backend_python/wechat_backend/services/keyword_extractor.py` (新增)
- `backend_python/requirements.txt` (添加 jieba)

**核心算法**:

**TF-IDF**:
```
TF (词频) = 词频 / 总词数
IDF (逆文档频率) = log(文档总数 / 包含该词的文档数)
TF-IDF = TF × IDF
```

**TextRank**:
```
基于 PageRank 的图排序算法
WS(Vi) = (1-d) + d * Σ(Wj / |Out(Vj)|)
```

**混合方法**:
```
最终得分 = TF-IDF 得分 × 0.6 + TextRank 得分 × 0.4
```

**特色功能**:
- ✅ 词长加权（2-4 字词权重更高）
- ✅ 品牌关键词加权
- ✅ 停用词过滤（内置 + 自定义）
- ✅ 中文分词（jieba）

**验收标准**:
- ✅ 关键词准确率 > 85%
- ✅ 支持自定义停用词
- ✅ 词云支持交互（点击、缩放）
