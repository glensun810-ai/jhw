七、API 定义
7.1 核心诊断 API (API_Spec_v2.0.json)
POST /api/v2/test/submit
  提交品牌 AI 诊断任务
  Request:
    - brand_list: string[]      # 监控品牌列表
    - selectedModels: string[]  # 启用的 AI 模型 ID
    - customQuestions: string[] # 自定义测试问题
  Response (202):
    - task_id: UUID
    - message: string

GET /api/v2/test/status/{task_id}
  轮询任务进度与分阶段状态
  Response (200):
    - task_id: string
    - progress: 0-100
    - stage: init | ai_fetching | ranking_analysis | source_tracing | completed
    - status_text: string
    - is_completed: boolean
    - created_at: datetime

GET /api/v2/test/result/{task_id}
  获取诊断任务完整结果
  Response (200): DeepIntelligenceResult
    - exposure_analysis:
      - ranking_list: string[]
      - brand_details: {brand: {rank, word_count, sov_share, sentiment_score}}
      - unlisted_competitors: string[]
    - source_intelligence:
      - source_pool: [{id, url, site_name, citation_count, domain_authority}]
      - citation_rank: string[]
    - evidence_chain:
      - [{negative_fragment, associated_url, source_name, risk_level}]
    - comparative_analysis:
      - common_keywords: string[]
      - my_brand_unique_keywords: string[]
      - competitor_unique_keywords: string[]
      - differentiation_gap: string
