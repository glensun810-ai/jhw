## 📊 问题描述

### 用户报告

从诊断记录中，点击查看历史诊断报告详情页时，无法打开，提示：
> "模拟器长时间没有响应，请确认你的业务逻辑中是否有复杂运算，或者死循环。"

### 问题分析

经过代码审查，发现两处死循环风险点：

---

## 🔍 根因分析

### 死循环点 1: diagnosis.js

**位置**: `miniprogram/pages/diagnosis/diagnosis.js` Line 600

**问题代码**:
```javascript
wx.showModal({
  title: '加载失败',
  content: error.message || '无法加载历史报告，请稍后重试',
  confirmText: '重试',
  cancelText: '取消',
  success: (res) => {
    if (res.confirm) {
      this.loadHistoryReport(reportId);  // ❌ 递归调用导致死循环
    } else {
      wx.navigateBack();
    }
  }
});
```

**问题**: 用户点击"重试"时，递归调用 `loadHistoryReport`，如果 API 持续失败，会导致无限递归。

## 🔧 修复方案

### 修复 1: diagnosis.js

**修改内容**:
1. 添加 `_isLoadingHistory` 标志防止重复加载
2. 重试时使用 `currentPage.onLoad()` 而非递归调用

**修复代码**:
```javascript
async loadHistoryReport(reportId) {
  // 【死循环修复 - 2026-03-14】防止重复加载
  if (this._isLoadingHistory) {
    console.warn('[DiagnosisPage] ⚠️ 正在加载中，跳过重复请求');
    return;
  }
  
  showLoading('加载中...');
  this._isLoadingHistory = true;

  try {
    // ... 加载逻辑 ...
  } catch (error) {
    hideLoading();
    this._isLoadingHistory = false;

    // 【死循环修复 - 2026-03-14】使用页面导航而非递归调用
    wx.showModal({
      title: '加载失败',
      content: error.message || '无法加载历史报告，请稍后重试',
      showCancel: true,
      confirmText: '重试',
      cancelText: '返回',
      success: (res) => {
        if (res.confirm) {
          // 【修复】使用页面重新加载而非递归调用
          const pages = getCurrentPages();
          const currentPage = pages[pages.length - 1];
          if (currentPage && currentPage.options && currentPage.options.reportId) {
            currentPage.onLoad(currentPage.options);
          }
        } else {
          wx.navigateBack();
        }
      }
    });
  }
}
```

## 🎓 经验总结

### 问题根因

在错误处理对话框的"重试"按钮中，直接递归调用原方法，导致：
1. 每次重试都增加调用栈深度
2. 如果 API 持续失败，调用栈无限增长
3. 最终导致模拟器/真机长时间无响应

### 修复原则

1. **防止重复加载**: 使用标志位 `_isLoadingHistory`
2. **避免递归调用**: 使用 `currentPage.onLoad()` 而非 `this.loadHistoryReport()`
3. **清理状态**: 在 finally 块或 catch 块中清理标志位

### 最佳实践

```javascript
// ❌ 错误：递归调用
wx.showModal({
  success: (res) => {
    if (res.confirm) {
      this.someMethod();  // 递归调用
    }
  }
});

// ✅ 正确：使用页面生命周期方法
wx.showModal({
  success: (res) => {
    if (res.confirm) {
      const pages = getCurrentPages();
      const currentPage = pages[pages.length - 1];
      currentPage.onLoad(currentPage.options);  // 重新加载页面
    }
  }
});
```
