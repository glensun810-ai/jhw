## 🎓 技术亮点

### 1. 算法优化

**TF-IDF + TextRank 混合关键词提取**:
```
最终得分 = TF-IDF 得分 × 0.6 + TextRank 得分 × 0.4
词长加权：2-4 字词权重 +50%
品牌关键词加权：+30%
```

**6 维度情感分析**:
```
喜悦、愤怒、悲伤、恐惧、惊讶、厌恶
程度副词加权：high(1.5x), medium(1.2x), low(0.8x)
否定词检测：情感反转 -50%
```

### 2. 架构设计

**服务化设计**: 每个功能独立服务，便于维护和扩展
**降级方案**: WebSocket 失败自动降级到 HTTP 轮询
**缓存机制**: 7 天 TTL，自动过期清理

### 3. 用户体验

**报告导出**: 支持 Excel 二次分析，长图分享传播
**问题模板**: 11 个行业模板，覆盖主流场景
**版本对比**: 同品牌多报告对比，差异高亮显示
