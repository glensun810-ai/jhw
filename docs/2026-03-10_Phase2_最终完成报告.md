### 2.2 混合模式设计

**API + Local 混合**:
- 优先使用 API 数据
- API 失败时降级到本地数据
- 支持离线使用

**实现**:
```javascript
async loadBrandCompare() {
  try {
    const res = await analyticsApi.getBrandCompareApi();
    this.setData({ brandCompareData: res.data.brands });
  } catch (error) {
    // 降级到本地数据
    this.loadBrandCompareLocal();
  }
}
```
