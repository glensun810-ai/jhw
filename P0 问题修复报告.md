## 一、问题根因确认

### 1.1 核心问题

**问题**: AI 调用结果未保存到 `diagnosis_results` 表

**根因**: 
- `nxm_execution_engine.py` 中只调用了 `save_dimension_result()` 
- 该函数保存到 `dimension_results` 表，而非 `diagnosis_results` 表
- 导致前端查询 `diagnosis_results` 时得到空结果

### 1.2 代码位置

**文件**: `backend_python/wechat_backend/nxm_execution_engine.py`

**问题代码** (第 479-488 行):
```python
save_dimension_result(
    execution_id=execution_id,
    dimension_name=f"{main_brand}-{model_name}",
    dimension_type="ai_analysis",
    source=model_name,
    status=dim_status,
    score=dim_score,
    data=geo_data if dim_status == "success" else None,
    error_message=ai_result.error_message if dim_status == "failed" else (parse_error if parse_error else None)
)
```

**缺失的代码**:
```python
# 【P0 关键修复】保存 AI 调用结果到 diagnosis_results 表
if dim_status == "success" and geo_data:
    try:
        from wechat_backend.diagnosis_report_repository import DiagnosisResultRepository
        
        diagnosis_result = {
            'brand': main_brand,
            'question': question,
            'model': model_name,
            'response': {
                'content': str(ai_result.data),
                'latency': ai_result.latency
            },
            'geo_data': geo_data,
            'status': 'success',
            'error': None,
            'quality_score': dim_score,
            'quality_level': 'high' if dim_score >= 80 else 'medium'
        }
        
        result_repo = DiagnosisResultRepository()
        result_id = result_repo.add_by_execution_id(execution_id, diagnosis_result)
        
        api_logger.info(f"[NxM] ✅ diagnosis_results 保存成功：{execution_id}")
        
    except Exception as save_err:
        api_logger.error(f"[NxM] ⚠️ diagnosis_results 保存失败：{save_err}")
```

## 二、修复方案

### 2.1 手动修复步骤

**步骤 1**: 打开文件 `backend_python/wechat_backend/nxm_execution_engine.py`

**步骤 2**: 找到第 479-488 行（`save_dimension_result` 调用）

**步骤 3**: 在 `save_dimension_result(...)` 调用之后，`save_task_status(...)` 调用之前，插入以下代码：

```python
# 【P0 关键修复】保存 AI 调用结果到 diagnosis_results 表
if dim_status == "success" and geo_data:
    try:
        from wechat_backend.diagnosis_report_repository import DiagnosisResultRepository
        
        diagnosis_result = {
            'brand': main_brand,
            'question': question,
            'model': model_name,
            'response': {
                'content': str(ai_result.data) if hasattr(ai_result, 'data') else str(ai_result),
                'latency': ai_result.latency if hasattr(ai_result, 'latency') else None
            },
            'geo_data': geo_data,
            'status': 'success',
            'error': None,
            'quality_score': dim_score,
            'quality_level': 'high' if dim_score and dim_score >= 80 else 'medium' if dim_score and dim_score >= 60 else 'low'
        }
        
        result_repo = DiagnosisResultRepository()
        result_id = result_repo.add_by_execution_id(execution_id, diagnosis_result)
        
        api_logger.info(f"[NxM] ✅ diagnosis_results 保存成功：execution_id={execution_id}, result_id={result_id}")
        
    except Exception as save_err:
        api_logger.error(f"[NxM] ⚠️ diagnosis_results 保存失败：execution_id={execution_id}, 错误：{save_err}")
```

**步骤 4**: 保存文件并重启后端服务

## 三、其他 P0 问题修复

### 3.1 P0-2: AI 响应日志缺失

**问题**: `logs/ai_responses.log` 为空

**原因**: AIResponseLogger 未被调用

**修复位置**: `nxm_execution_engine.py`

**修复代码**: 在 AI 调用成功后添加日志记录：

```python
# 【P0-2 修复】记录 AI 响应日志
try:
    from wechat_backend.utils.ai_response_logger import log_ai_response
    
    log_ai_response(
        question=question,
        response=str(ai_result.data) if hasattr(ai_result, 'data') else str(ai_result),
        platform=model_name,
        model=model_name,
        brand=main_brand,
        latency_ms=int(ai_result.latency * 1000) if hasattr(ai_result, 'latency') else None,
        success=ai_result.success,
        error_message=ai_result.error_message if not ai_result.success else None
    )
    
    api_logger.info(f"[AI 日志] ✅ 已记录：{execution_id}-{question[:20]}")
    
except Exception as log_err:
    api_logger.error(f"[AI 日志] ⚠️ 记录失败：{log_err}")
```
