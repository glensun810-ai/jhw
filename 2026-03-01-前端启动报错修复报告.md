## 问题分析

### 根本原因
在 `loadPlatformConfigStatus` 函数中，代码直接调用 `that.data.domesticAiModels.map()`，但没有检查 `domesticAiModels` 是否已初始化为数组。

### 执行时序问题
```javascript
// onLoad 中的执行顺序
onLoad: function (options) {
  // 1. 初始化默认值
  initializeDefaults(this);
  
  // 2. 检查服务器连接（异步）
  checkServerConnection(this);
  
  // 3. 初始化 AI 平台数据（同步）
  this.initDomesticAiModels();      // ← 需要时间执行
  this.initOverseasAiModels();      // ← 需要时间执行
  
  // 4. 加载用户偏好（异步）
  loadUserPlatformPreferences(this);
  
  // 5. 读取高级设置状态
  // ...
  
  // 6. 获取平台配置状态 ← 可能先于步骤 3 完成！
  this.loadPlatformConfigStatus();
}
```

### 问题细节
1. `initDomesticAiModels()` 和 `initOverseasAiModels()` 通过 `setData` 异步更新数据
2. `loadPlatformConfigStatus()` 立即调用，可能在 `setData` 完成前执行
3. 此时 `that.data.domesticAiModels` 仍然是 `undefined` 或初始值
4. 对非数组调用 `.map()` 导致报错

## 修复方案

### 1. 添加数组类型检查
```javascript
loadPlatformConfigStatus: function() {
  const that = this;
  
  // ✅ 防御性检查：确保 domesticAiModels 是数组
  if (!Array.isArray(that.data.domesticAiModels)) {
    console.warn('[平台配置] domesticAiModels 不是数组，等待初始化完成');
    setTimeout(() => {
      this.loadPlatformConfigStatus();
    }, 100);
    return;
  }
  
  // ✅ 防御性检查：确保 overseasAiModels 是数组

    setTimeout(() => {
      this.loadPlatformConfigStatus();
    }, 100);
    return;
  }
  
  // ... 继续执行
}
```

## 修复后的完整代码

```javascript
/**
 * 【新增】加载平台配置状态，标记未配置的平台
 */
loadPlatformConfigStatus: function() {
  const that = this;
  
  // 防御性检查：确保 domesticAiModels 和 overseasAiModels 是数组
  if (!Array.isArray(that.data.domesticAiModels)) {
    console.warn('[平台配置] domesticAiModels 不是数组，等待初始化完成');
    // 延迟重试
    setTimeout(() => {
      this.loadPlatformConfigStatus();
    }, 100);
    return;
  }
  
  if (!Array.isArray(that.data.overseasAiModels)) {
    console.warn('[平台配置] overseasAiModels 不是数组，等待初始化完成');
    // 延迟重试
    setTimeout(() => {
      this.loadPlatformConfigStatus();
    }, 100);
    return;
  }
  
  wx.request({
    url: getApp().globalData.serverUrl + '/api/platform-status',
    method: 'GET',
    success: function(res) {
      if (res.statusCode === 200 && res.data.status === 'success') {
        const platforms = res.data.platforms;

        // 更新国内平台配置状态（添加安全检查）
        const domesticAiModels = (that.data.domesticAiModels || []).map(model => {
          const platformStatus = platforms[model.id];
          return {
            ...model,
            isConfigured: platformStatus ? platformStatus.isConfigured : false,
            disabled: platformStatus ? !platformStatus.isConfigured : true
          };
        });

        // 更新海外平台配置状态（添加安全检查）
        const overseasAiModels = (that.data.overseasAiModels || []).map(model => {
          const platformStatus = platforms[model.id];
          return {
            ...model,
            isConfigured: platformStatus ? platformStatus.isConfigured : false,
            disabled: platformStatus ? !platformStatus.isConfigured : true
          };
        });

        that.setData({
          domesticAiModels,
          overseasAiModels
        });

        console.log('[平台配置] 已更新平台配置状态');
      }
    },
    fail: function(err) {
      console.warn('[平台配置] 获取平台状态失败:', err);
    }
  });
}
```
