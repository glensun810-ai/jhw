## 🔧 故障排查

### 问题 1: 仍然是 "Working outside of application context"

**解决方案：**
```bash
# 确保完全停止服务器
ps aux | grep python | grep run.py
kill -9 <所有相关进程>

# 清理缓存
find . -type d -name "__pycache__" -exec rm -rf {} + 2>/dev/null

# 重新启动
python3 run.py
```

### 问题 2: 没有 AI 调用日志

**可能原因：**
- AI API 密钥未配置
- AI 平台不可用
- 诊断在 AI 调用前就失败了

**检查：**
```bash
# 查看 AI 平台状态
curl http://127.0.0.1:5001/api/platform-status

# 检查 AI 配置
grep -r "DOUBAO_API_KEY\|AI_API_KEY" config/ .env*
```

### 问题 3: 数据库连接泄漏警告

**正常现象：** 如果看到：
```
[连接泄漏] 连接超时未归还
[连接泄漏] 强制归还连接
```

这是连接池监控在正常工作，会自动清理。
