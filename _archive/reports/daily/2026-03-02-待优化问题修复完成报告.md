```python
def _apply_frequency_control(self) -> float:
    """应用请求频率控制"""
    current_time = time.time()
    elapsed = current_time - self._last_request_time
    
    if elapsed < self._min_request_interval:
        wait_time = self._min_request_interval - elapsed
        api_logger.info(f"[Qwen] 频率控制：延迟 {wait_time:.2f}秒")
        time.sleep(wait_time)
    
    self._last_request_time = time.time()
```

### 2.3 WebSocket 异步线程事件循环修复 ✅

**文件**: `backend_python/wechat_backend/services/realtime_push_service.py`

**问题**:
```
[RealtimePush] 同步完成推送失败：There is no current event loop in thread 'DiagnosisThread-xxx'.
```

**根因**: 异步线程中没有事件循环

**修复前**:
```python
def send_complete_sync(...):
    loop = asyncio.get_event_loop()
    if loop.is_running():
        asyncio.create_task(...)
    else:
        loop.run_until_complete(...)
```

**修复后**:
```python
def send_complete_sync(...):
    # 在异步线程中创建新的事件循环
    loop = asyncio.new_event_loop()
    asyncio.set_event_loop(loop)
    try:
        loop.run_until_complete(service.send_complete(...))
    finally:
        loop.close()
```

**修复文件**:
- `send_progress_sync()`
- `send_complete_sync()`
- `send_error_sync()`
