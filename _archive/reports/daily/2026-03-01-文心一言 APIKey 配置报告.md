## 使用方法

### 在代码中使用
```python
from wechat_backend.ai_adapters.factory import AIAdapterFactory

# 获取文心一言适配器
adapter = AIAdapterFactory.get_adapter('wenxin')

# 调用 API
response = adapter.generate(prompt="你好，请介绍一下自己")
```

### 在 API 请求中使用
```bash
curl -X POST http://localhost:5001/api/diagnosis \
  -H "Content-Type: application/json" \
  -d '{
    "brand": "华为",
    "model": "wenxin",
    "question": "华为的手机怎么样？"
  }'
```

## 故障排查

### 问题 1: Provider 未注册
**症状**: 日志中出现 `Provider NOT registered for 'wenxin'`

**解决方案**:
1. 检查 `.env` 文件是否存在
2. 检查 `WENXIN_API_KEY` 是否正确配置
3. 重启应用

### 问题 2: API 调用失败
**症状**: 调用文心一言 API 时返回 401 或 403 错误

**解决方案**:
1. 验证 API Key 是否有效
2. 检查百度千帆控制台配额是否充足
3. 确认网络可以访问百度 API 端点
