        api_logger.info(f"[Qwen] 频率控制：延迟 {wait_time:.2f}秒")

### 问题 6: 结果页 500 错误重试策略

**问题描述**: 结果页请求报告遇到 500 错误，可能是数据库写入延迟

**修复建议**:

#### 前端延迟重试

**文件**: `miniprogram/pages/report-v2/report-v2.js`

**建议代码**:
```javascript
async fetchReport(executionId, retryCount = 0) {
  try {
    const report = await getReportApi(executionId);
    return report;
  } catch (error) {
    // 【P1 建议 - 2026-03-04】500 错误延迟重试
    if (error.statusCode === 500 && retryCount < 2) {
      console.warn(`[ReportPage] 500 错误，等待 500ms 后重试 (${retryCount + 1}/2)`);
      
      await new Promise(resolve => setTimeout(resolve, 500));
      
      return this.fetchReport(executionId, retryCount + 1);
    }
    
    throw error;
  }
}
```
