## 1. 产品概述

### 1.1 产品定位

**AI 品牌诊断系统**（品牌名称：进化湾 GEO / 云程企航）是一款基于微信小程序的企业级品牌分析 SaaS 产品。系统整合多个主流 AI 大模型（豆包、DeepSeek、通义千问、ChatGPT、Gemini、文心一言、智谱等），为企业提供智能化、多维度的品牌诊断、竞争情报分析、市场洞察等服务。

### 1.2 核心价值

- **多 AI 平台整合**: 一站式接入 7+ 主流 AI 大模型，智能调度，成本优化
- **深度品牌洞察**: 基于 AI 的品牌露出分析、信源归因、竞争格局研判
- **实时诊断反馈**: WebSocket 实时推送诊断进度，用户体验流畅
- **企业级安全**: JWT 认证、数据加密、SQL 注入防护、速率限制
- **智能监控告警**: 系统健康监控、异常告警、自动故障转移

### 1.3 技术栈

| 层级 | 技术选型 |
|------|----------|
| 前端 | 微信小程序原生开发 (WXML/WXSS/JavaScript) |
| 后端 | Python 3.14+ / Flask 2.3+ |
| 数据库 | SQLite3 (支持读写分离、WAL 模式) |
| 缓存 | Redis (可选) / 内存缓存 |
| 实时通信 | WebSocket / HTTP 轮询降级 |
| AI 平台 | 豆包/DeepSeek/通义千问/ChatGPT/Gemini/文心一言/智谱 |

## 2. 目标用户与使用场景

### 2.1 目标用户群体

| 用户类型 | 描述 | 核心需求 |
|----------|------|----------|
| **企业品牌管理者** | 品牌总监、市场经理 | 了解品牌市场表现、竞争格局、优化品牌策略 |

### 2.2 核心使用场景

#### 场景一：品牌诊断
1. 用户进入小程序首页
2. 输入目标品牌名称（支持多品牌对比）
3. 选择 AI 模型（支持多模型组合）
4. 选择/自定义诊断问题
5. 启动诊断，实时查看进度
6. 获取诊断报告，查看分析结果

1. 用户进入"历史记录"页面
2. 查看个人历史诊断列表
3. 点击查看详情或下载报告
4. 支持收藏重要报告

#### 场景三：管理后台
1. 管理员登录后台
2. 查看用户统计、诊断统计
3. 管理系统配置、AI 平台状态
4. 查看系统监控、处理告警

### 4.1 品牌诊断功能

#### 4.1.1 功能描述

品牌诊断是系统的核心功能，通过 AI 模型对指定品牌进行多维度分析，生成包含品牌露出、信源归因、竞争分析等内容的深度报告。

#### 4.1.2 业务流程

```
用户输入 → 前端验证 → API 调用 → 后端处理 →
AI 调用 → 结果聚合 → 报告生成 → 返回前端 → 展示结果
```

#### 4.1.4 诊断编排器 (DiagnosisOrchestrator)

诊断编排器负责协调诊断流程的所有环节，确保数据一致性和完整性。

**核心职责**:
1. API 响应保存 - 确保原始响应持久化
2. 统计分析 - 生成统计指标
3. 结果聚合 - 聚合多模型结果
4. 报告生成 - 生成最终报告
5. 状态同步 - 内存和数据库状态一致

**执行流程**:
```python
class DiagnosisOrchestrator:
    def execute(self, config):
        # 1. 保存 API 响应
        self._save_api_response(config)
        
        # 2. 执行 AI 调用
        ai_results = self._execute_ai_calls(config)
        
        # 3. 统计分析
        stats = self._analyze_statistics(ai_results)
        
        # 4. 结果聚合
        aggregated = self._aggregate_results(ai_results)
        
        # 5. 报告生成
        report = self._generate_report(aggregated, stats)
        
        # 6. 状态更新
        self._update_status('completed')
        
        return report
```

```javascript
 * 轮询管理器 - 负责 HTTP 轮询降级
 */
class PollingManager {
  startPolling(executionId, callbacks) {
    this.executionId = executionId;
    this.callbacks = callbacks;
    this.interval = POLLING_CONFIG.initialInterval;
    this._poll();
  }
  
  _poll() {
    if (this._shouldStop()) return;
    
    // 调用状态查询 API
    api.getDiagnosisStatus(this.executionId)
      .then((status) => {
        this.callbacks.onProgress(status);
        if (status.is_completed) {
          this.callbacks.onComplete(status);
        } else {
          // 指数退避
          this.interval *= POLLING_CONFIG.backoffMultiplier;
          this.timer = setTimeout(() => this._poll(), this.interval);
        }
      });
  }
}
```

```python
class DiagnosisOrchestrator:
    """
    诊断编排器 - 统一编排诊断流程
    
    核心原则:
    1. 顺序执行 - API 响应保存 → 统计分析 → 结果聚合 → 报告生成
    2. 状态一致 - 内存和数据库原子性更新
    3. 完整持久化 - 所有结果必须完整保存
    4. 统一调度 - 协调所有子流程
    """
    
    def __init__(self, execution_id, config):
        self.execution_id = execution_id
        self.config = config
        self.state_manager = StateManager()
        self.result_validator = ResultValidator()
    
    def execute(self):
        """执行诊断流程"""
        try:
            # 1. 保存 API 响应
            self._save_api_response()
            
            # 2. 执行 AI 调用
            ai_results = self._execute_ai_calls()
            
            # 3. 统计分析
            stats = self._analyze_statistics(ai_results)
            
            # 4. 结果聚合
            aggregated = self._aggregate_results(ai_results)
            
            # 5. 报告生成
            report = self._generate_report(aggregated, stats)
            
            # 6. 验证结果
            self.result_validator.validate(report)
            
            # 7. 更新状态
            self._update_status('completed', report)
            
            # 8. 发送完成通知
            send_complete_sync(self.execution_id, report)
            
            return report
            
        except Exception as e:
            self._handle_error(e)
            raise
```

```python
class DoubaoAdapter(AIClient):
    """豆包 AI 适配器"""
    
    def __init__(self, api_key, model=None):
        super().__init__(api_key, model or 'doubao-seed-1-8-251228')
        self.base_url = 'https://ark.cn-beijing.volces.com/api/v3'
        self.model_priority = self._load_model_priority()
    
    def chat(self, prompt: str) -> AIResponse:
        """发送聊天请求"""
        headers = {
            'Authorization': f'Bearer {self.api_key}',
            'Content-Type': 'application/json'
        }
        
        data = {
            'model': self._select_model(),
            'messages': [{'role': 'user', 'content': prompt}]
        }
        
        response = self.session.post(
            f'{self.base_url}/chat/completions',
            headers=headers,
            json=data,
            timeout=self.timeout
        )
        
        return self._parse_response(response)
    
    def _select_model(self) -> str:
        """自动选择最高优先级可用模型"""
        for model in self.model_priority:
```

```python
class JWTManager:
    """JWT 令牌管理器"""
    
    def generate_token(self, user_id, additional_claims=None) -> str:
        """生成 JWT 令牌"""
        payload = {
            'user_id': user_id,
            'exp': datetime.utcnow() + timedelta(hours=24),
            'iat': datetime.utcnow()
        }
        if additional_claims:
            payload.update(additional_claims)
        return jwt.encode(payload, SECRET_KEY, algorithm='HS256')
    
    def verify_token(self, token) -> dict:
        """验证 JWT 令牌"""
        try:
            payload = jwt.decode(token, SECRET_KEY, algorithms=['HS256'])
            return payload
        except jwt.ExpiredSignatureError:
            raise TokenExpiredError()
        except jwt.InvalidTokenError:
            raise TokenInvalidError()
```

### 9.5 数据加密

#### 9.5.1 数据加密服务

```python
class DataEncryptionService:
    """数据加密服务"""
    
    def __init__(self, encryption_key):
        self.encryption_key = encryption_key
        self.cipher = Cipher(algorithms.AES(encryption_key), modes.CBC(encryption_key[:16]))
    
    def encrypt(self, data) -> str:
        """加密数据"""
        # PKCS7 填充
        padding_length = 16 - (len(data) % 16)
        data += chr(padding_length) * padding_length
        
        # 加密
        encryptor = self.cipher.encryptor()
        encrypted = encryptor.update(data.encode()) + encryptor.finalize()
        
        return base64.b64encode(encrypted).decode()
    
    def decrypt(self, encrypted_data) -> str:
        """解密数据"""
        # 解密
        encrypted = base64.b64decode(encrypted_data)
        decryptor = self.cipher.decryptor()
        decrypted = decryptor.update(encrypted) + decryptor.finalize()
        
        # 去除填充
        padding_length = decrypted[-1]
        return decrypted[:-padding_length].decode()
```

```python
class MemoryCache:
    """内存缓存"""
    
    def __init__(self, ttl=300):
        self.cache = {}
        self.ttl = ttl
    
    def get(self, key):
        """获取缓存"""
        if key in self.cache:
            value, expire_time = self.cache[key]
            if time.time() < expire_time:
                return value
            else:
                del self.cache[key]
        return None
    
    def set(self, key, value, ttl=None):
        """设置缓存"""
        expire_time = time.time() + (ttl or self.ttl)
        self.cache[key] = (value, expire_time)
```

#### 10.5.2 读写路由

```python
class DatabaseRouter:
    """数据库路由器"""
    
    def __init__(self, master_path, slave_paths):
        self.master = sqlite3.connect(master_path)
        self.slaves = [sqlite3.connect(path) for path in slave_paths]
        self.current_slave_index = 0
    
    def get_write_connection(self):
        """获取写连接"""
        return self.master
    
    def get_read_connection(self):
        """获取读连接（轮询）"""
        slave = self.slaves[self.current_slave_index]
        self.current_slave_index = (self.current_slave_index + 1) % len(self.slaves)
        return slave
```
