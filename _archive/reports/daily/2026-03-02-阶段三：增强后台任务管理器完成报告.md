## 一、实施内容

### 1.1 新增数据结构

#### TaskStatus 枚举
```python
class TaskStatus(Enum):
    """任务状态枚举"""
    PENDING = "pending"
    RUNNING = "running"
    COMPLETED = "completed"
    FAILED = "failed"
    TIMEOUT = "timeout"
```

#### AnalysisTask 数据类
```python
@dataclass
class AnalysisTask:
    """分析任务记录"""
    task_id: str
    execution_id: str
    task_type: str  # 'brand_analysis', 'competitive_analysis', 'statistics'
    payload: Dict[str, Any]
    status: TaskStatus = TaskStatus.PENDING
    created_at: datetime = field(default_factory=datetime.now)
    started_at: Optional[datetime] = None
    completed_at: Optional[datetime] = None
    result: Optional[Dict[str, Any]] = None
    error: Optional[str] = None
    timeout_seconds: int = 300  # 默认 5 分钟超时
```

### 1.2 核心功能实现

#### 1.2.1 分析任务提交方法

**方法**: `submit_analysis_task()`

**功能**:
- 生成唯一任务 ID（格式：`{execution_id}_{task_type}_{uuid}`）
- 创建任务记录并保存到队列
- 提交到线程池异步执行
- 支持自定义超时时间

**代码位置**: `background_service_manager.py:303-344`

**使用示例**:
```python
manager = get_background_service_manager()

# 提交品牌分析任务
task_id = manager.submit_analysis_task(
    execution_id='exec-001',
    task_type='brand_analysis',
    payload={
        'results': ai_results,
        'user_brand': '华为',
        'competitor_brands': ['小米', '苹果']
    },
    timeout_seconds=300
)
```

---

#### 1.2.2 任务执行引擎

**方法**: `_execute_analysis_task()`

**功能**:
- 更新任务状态为 `RUNNING`
- 记录任务开始时间
- 根据任务类型分发到不同执行器
- 捕获异常并更新状态
- 处理超时情况

**任务类型分发**:
```python
if task_type == 'brand_analysis':
    result = self._execute_brand_analysis(payload)
elif task_type == 'competitive_analysis':
    result = self._execute_competitive_analysis(payload)
elif task_type == 'statistics':
    result = self._execute_statistics(payload)
else:
    raise ValueError(f"未知任务类型：{task_type}")
```

### 3.2 设计亮点

1. **异步非阻塞**: 分析任务使用线程池异步执行，不阻塞主流程
2. **状态完整追踪**: 记录任务从创建到完成的完整生命周期
3. **超时保护**: 每个任务都有独立的超时计时器
4. **错误隔离**: 单个任务失败不影响其他任务
5. **可清理**: 支持自动清理历史任务记录，避免内存泄漏
6. **类型安全**: 使用 Enum 和 dataclass 确保类型安全

class DiagnosisOrchestrator:

### 4.2 前端轮询分析进度

```javascript
// 前端 JavaScript 代码示例
async function waitForAnalysisComplete(executionId, timeoutSeconds = 120) {
    const startTime = Date.now();
    
    while ((Date.now() - startTime) / 1000 < timeoutSeconds) {
        const response = await fetch(`/api/analysis-status/${executionId}`);
        const status = await response.json();
        
        if (status.analysis_complete) {
            return status.analysis_results;
        }
        
        // 每 2 秒检查一次
        await new Promise(resolve => setTimeout(resolve, 2000));
    }
    
    throw new Error('分析超时');
}
```
