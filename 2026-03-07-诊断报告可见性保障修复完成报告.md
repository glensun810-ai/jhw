# 诊断报告可见性保障修复完成报告

**修复日期**: 2026-03-07  
**修复优先级**: P0（最高优先级）  
**修复类型**: 用户体验阻断问题  
**影响范围**: 所有诊断失败场景  

---

## 一、问题描述

### 1.1 原始问题
用户反馈：最新运行后，前端提示"无有效结果"，无法查看详细诊断报告结果。

### 1.2 问题根因
**核心问题**: 系统在任何情况下，只要诊断结果为空（即使有执行元数据），就完全阻断用户查看报告。

**具体表现**:
1. **后端**: 当所有 AI 调用失败时，返回 `success: false` 和空结果数组，不返回任何元数据
2. **前端 index.js**: 检测到 `results.length === 0` 时弹出"无有效结果"模态框，阻止用户查看报告
3. **前端 results.js**: 同样逻辑阻止用户查看任何诊断信息

**用户体验影响**:
- ❌ 即使有诊断配置信息（AI 模型、诊断问题），用户也无法查看
- ❌ 即使有执行日志（发生了什么、哪里失败），用户也无法查看
- ❌ 即使有错误详情（具体错误信息），用户也无法查看
- ❌ 用户完全不知道诊断过程中发生了什么

---

## 二、修复方案

### 2.1 核心原则
**任何情况下，都必须让用户看到尽可能多的信息**

即使诊断完全失败，用户也应该看到：
- ✅ 诊断配置（使用的 AI 模型、诊断问题）
- ✅ 执行日志（发生了什么、哪里失败了）
- ✅ 错误详情（具体错误信息）
- ✅ 恢复建议（如何解决问题）

**新增函数**: `showFailedStateWithDetails()`

```javascript
/**
 * 【P0 关键修复 - 2026-03-07】新增：展示失败详情（配置回顾 + 执行日志 + 错误详情）
 * 即使诊断完全失败，也让用户看到发生了什么
 */
showFailedStateWithDetails: function(resultData, brandName) {
  // 提取元数据
  const executionMetadata = resultData.execution_metadata || {};
  const errorDetails = resultData.error_details || resultData.error || {};
  const selectedModels = executionMetadata.selected_models || resultData.selected_models || [];
  const customQuestions = executionMetadata.custom_questions || resultData.custom_questions || [];
  const executionLog = executionMetadata.execution_log || [];
  
  // 构建错误日志列表
  const errorLogList = executionLog.map((log, index) => ({
    id: index,
    task: log.task_id || `任务${index + 1}`,
    errorType: this._extractErrorType(log.error || ''),
    errorMessage: log.error || '未知错误',
    status: log.status || 'failed',
    timestamp: log.timestamp || new Date().toISOString()
  }));

  // 设置页面数据
  this.setData({
    // 基础信息
    targetBrand: brandName || executionMetadata.main_brand || '',
    selectedModels: selectedModels,
    customQuestions: customQuestions,
    
    // 执行统计
    totalTasks: totalTasks,
    completedTasks: completedTasks,
    completionRate: completionRate,
    
    // 错误详情
    errorLogList: errorLogList,
    
    // 失败状态
    showEmptyState: true,
    emptyTitle: '诊断失败',
    emptyMessage: errorDetails.message || '诊断过程中遇到错误',
    
    // 新增：显示配置回顾和执行日志
    showConfigSection: true,
    showErrorDetailsModal: false
  });
  
  // 显示非阻断式提示
  wx.showToast({
    title: '诊断失败，请查看详情',
    icon: 'none',
    duration: 3000
  });
}

/**
 * 【P0 关键修复 - 2026-03-07】跳转到报告页（即使无结果也展示详情）
 */
const navigateToReportPage = (executionId, options = {}) => {
  const {
    hasResults = false,
    showExecutionLog = true,
    showConfigReview = true
  } = options;

  // 保存元数据到 Storage
  wx.setStorageSync('diagnosis_metadata_' + executionId, {
    hasResults: hasResults,
    showExecutionLog: showExecutionLog,
    showConfigReview: showConfigReview,
    timestamp: Date.now()
  });

  // 跳转到报告页
  PageTransition.navigateTo(
    `/pages/results/results?executionId=${executionId}&showDetails=true`,
    ANIMATION_TYPES.SLIDE_LEFT
  );
};
```
