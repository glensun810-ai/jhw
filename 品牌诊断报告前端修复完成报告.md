## 三、数据流向

### 3.1 完整数据流

```
后端 API 响应
    ↓
{
  "brandAnalysis": {...},
  "userBrandAnalysis": {...},
  "competitorAnalysis": [...],
  "comparison": {...},
  "top3Brands": [...]
}
    ↓
dashboard/index.js - processServerData()
    ↓
setData({
  brandAnalysis,
  userBrandAnalysis,
  competitorAnalysis,
  comparison,
  top3Brands
})
    ↓
WXML 数据绑定
    ↓
品牌对比分析组件 + TOP3 竞品榜单组件
    ↓
用户可视化展示
```

### 3.2 条件渲染

| 组件 | 渲染条件 | 数据来源 |
|------|---------|---------|
| 品牌对比分析 | `brandAnalysis != null` | dashboard.brandAnalysis |
| 用户品牌分析卡 | `userBrandAnalysis != null` | dashboard.userBrandAnalysis |
| 竞争优势 | `comparison.advantages.length > 0` | dashboard.comparison |
| 竞品分析列表 | `competitorAnalysis.length > 0` | dashboard.competitorAnalysis |
| TOP3 榜单 | `top3Brands.length > 0` | dashboard.top3Brands |
